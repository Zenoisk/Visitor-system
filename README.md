# Visitor Management System

Visitor Management System is a web-based system developed during my internship to manage visitor registration and records.

## Features

- Visitor registration
- Admin dashboard
- Visitor record management
- User authentication
- Visitor status tracking
- Report/list view

- MySQL
- Blade
- HTML
- CSS
- JavaScript
